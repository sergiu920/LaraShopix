# LaraShopix 🛍️

A modern internet catalog built with the **Laravel + Inertia.js + React** stack.
Featuring an admin panel for managing products and categories.
Designed as a full-stack SPA-like experience powered by server-side routing — no separate API needed.

## Tech Stack

- **Backend:** Laravel 13.18.0
- **Frontend:** React + TypeScript (via Inertia.js)
- **Styling:** Tailwind CSS
- **Database:** MySQL

## Catalog Structure

The catalog is organized into the following categories:

- 💻 Notebooks
- 📱 Phones  
- ⌚ Smartwatches
